HTML tags

tag description
<hgroup></hgroup> Semantic. Denotes a group of <h*> tags, e.g. in the head of an article page.
<img> alt="..." </img> New method to add the descripton of the image. Keep in mind that alt text will be read by accessibility software!
<code></code> Displays its contents styled in a fashion intended to indicate that the text is a short fragment of computer code.
<section></section> It's a logical divisor that works exactly like <div> but in a semantic way. It wraps a section of element in a meaningful way
<span></span> Logical in-line divisor.
<video></video> Embed a video in the page. Text between tags will be displayed if the browser is unable to load the video. Mandatory attributes are:
  • src (instead of href): points to the video by its url;
  • width and height;
  • controls: displays player controls.
