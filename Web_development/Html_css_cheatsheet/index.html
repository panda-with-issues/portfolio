HTML & CSS cheatsheet

No more google-fu for hard-to-remember tags and properties

HTML tags

tag description
<a target="_blank"></a> Open the link inside a new browser tab
<header></header> Semantic divisor that contains the header of a page.
<hgroup></hgroup> Semantic. Denotes a group of <h*> tags, e.g. in the head of an article page.
<footer></footer> Semantic divisor ehich holds the footer parts of a page
<img alt="..." /> New method to add the descripton of the image. Keep in mind that alt text will be read by accessibility software!
<code></code> Displays its contents styled in a fashion intended to indicate that the text is a short fragment of computer code.
<main></main> Semantic divisor which contain the main informations of the page.
<nav></nav> Semantic division for the site's navigation menu.
<section></section> It's a logical divisor that works exactly like <div> but in a semantic way. It wraps a section of element in a meaningful way
<span></span> Logical in-line divisor.
<tbody></tbody> Semantic table divisor which contains the data of the table.
<td colspan="n" rowspan="n"></td> These attributes let the cell be multicolumn or multirow. They take an integer greater then 1 as value that represent the number or columns or rows the cell will span.
<tfoot></tfoot> Semantic table divisor which contains the eventual footer of the table.
<th scope="..."></th> Make the header either of a row or a column of the table. The attribute scope takes consequently the values of row or col.
<thead></thead> Semantic table divisor which contains the header of the table.
<video></video> Embed a video in the page. Text between tags will be displayed if the browser is unable to load the video. Mandatory attributes are:
  • src (instead of href): points to the video by its url;
  • width and height;
  • controls: displays player controls.

CSS properties

property description
opacity Set the opacity of the element selected. Values must be between 0 (invisible) and 1 (normal). The short form is allowed, e.g. if we wanto to give a value of 50% opacity, we can give .5 as value.
background-image: url(path) Set an image as background of the element selected. path must be the image's URL.
